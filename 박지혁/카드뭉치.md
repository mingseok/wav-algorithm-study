## 카드뭉치
## https://school.programmers.co.kr/learn/courses/30/lessons/159994

```python
def solution(cards1, cards2, goal):
    for go in goal:
        if cards1 and go == cards1[0]:
            cards1.pop(0)
        elif cards2 and go == cards2[0]:
            cards2.pop(0)
        else:
            return "No"
    return "Yes"
```

시간복잡도는 O(n**2)
