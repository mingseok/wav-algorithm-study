## 기능개발
## https://school.programmers.co.kr/learn/courses/30/lessons/42586

```python
import math
def solution(progresses, speeds):
    imsi2 = []
    stack = []
    answer = []

    for p, s in zip(progresses, speeds):
        remain = 100 - p
        imsi2.append((remain + s - 1) // s)  
    for im in imsi2:
        if not stack:                 
            stack.append(im)
        elif im <= stack[0]:         
            stack.append(im)
        else:                         
            answer.append(len(stack)) 
            stack = [im]              

    answer.append(len(stack))        
    return answer
```
n = 기능 개수 -> 시간복잡도는 O(n)
