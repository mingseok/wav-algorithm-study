## 귤 고르기
## https://school.programmers.co.kr/learn/courses/30/lessons/138476

```python
def solution(k, tangerine):
    count = {}
    for size in tangerine:
        if size in count:
            count[size] += 1
        else:
            count[size] = 1

    counts = list(count.values())
    counts.sort(reverse=True)

    total = 0  
    kinds = 0   

    for c in counts:
        total += c
        kinds += 1
        if total >= k:
            break

    return kinds
```

시간복잡도 : O(n log n)
