## 소수 찾기
## https://school.programmers.co.kr/learn/courses/30/lessons/12921

```python
def solution(n):
    prime = [True] * (n + 1)
    prime[0] = prime[1] = False

    for i in range(2, int(n**0.5) + 1):
        if prime[i]:
            for j in range(i*i, n+1, i):
                prime[j] = False

    return sum(prime)
```
시간복잡도 : O(n)
