```python
from itertools import combinations
def solution(number):
    answer = 0
    for a, b, c in combinations(number, 3):
        if a + b + c == 0:
            answer = answer + 1
    return answer
```

파이썬 공식문서를 보고, itertools 라이브러리의 combinations 메서드 사용.

n개중 k개를 뽑는 조합의 경우, 시간복잡도는 O(n**k)

위 코드는 n개중 3개를 뽑으니, 시간복잡도는 O(n**3)
