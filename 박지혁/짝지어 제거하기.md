## 짝지어 제거하기
## https://school.programmers.co.kr/learn/courses/30/lessons/12973

```python
def solution(s):
    stack = []
    for s_1 in s:
        stack.append(s_1)
        if len(stack) >= 2 and stack[-1] == stack[-2]:
            stack.pop(-1)
            stack.pop(-1)
    if len(stack) == 0:
        answer = 1
    else :
        answer = 0
    return answer
```


시간복잡도 : O(n)
