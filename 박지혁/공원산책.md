## 공원산책
## https://school.programmers.co.kr/learn/courses/30/lessons/172928

```python
def solution(park, routes):
    h = len(park)
    w = len(park[0])
    print(h)
    print(w)
    r = 0
    c = 0
    for i in range(h):
        for j in range(w):
            if park[i][j] == "S":
                r, c = i, j
                break

    drdc = {
        "E": (0, 1),
        "W": (0, -1),
        "S": (1, 0),
        "N": (-1, 0),
    }

    for route in routes:
        d, t = route.split()
        t = int(t)
        dr, dc = drdc[d]

        nr, nc = r, c
        ok = True

        for x in range(t):
            nr += dr
            nc += dc
            if not (0 <= nr < h and 0 <= nc < w):
                ok = False
                break
            if park[nr][nc] == "X":
                ok = False
                break

        if ok:
            r, c = nr, nc

    return [r, c]
```

전체 입력크기가 n 일때 시간복잡도는 O(n)
