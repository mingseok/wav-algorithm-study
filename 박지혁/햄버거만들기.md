## 햄버거만들기
## https://school.programmers.co.kr/learn/courses/30/lessons/133502

```python
def solution(ingredient):
    stack = []
    answer = 0
    for i in ingredient:
        stack.append(i)
        if len(stack) >= 4 and i == 1:
            if stack[-1] == 1 and stack[-2] == 3 and stack[-3] == 2 and stack[-4] == 1:
                stack.pop()
                stack.pop()
                stack.pop()
                stack.pop()
                answer = answer + 1
    return answer
```

ingredient를 한 번 순회 → O(n)
