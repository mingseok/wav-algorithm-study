## 부족한 금액 계산하기
## https://school.programmers.co.kr/learn/courses/30/lessons/82612

```python
def solution(price, money, count):
    total = 0
    
    for i in range(1, count + 1):
        total += price * i
    
    if total <= money:
        return 0
    else:
        return total - money
```
시간복잡도 : O(n)
