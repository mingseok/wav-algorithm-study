## 괄호 회전하기
## https://school.programmers.co.kr/learn/courses/30/lessons/76502

```python
def solution(s):
    answer = 0
    n = len(s)

    pair = {
        ')': '(',
        ']': '[',
        '}': '{'
    }

    for i in range(n):
        rotated = s[i:] + s[:i]
        stack = []
        ok = True

        for ch in rotated:
            if ch in "([{":
                stack.append(ch)
            else:
                if not stack or stack[-1] != pair[ch]:
                    ok = False
                    break
                stack.pop()

        if ok and not stack:
            answer += 1

    return answer
```
시간복잡도 : O(n)
