```python
def solution(people, limit):
    boats = 0
    people.sort()
    i = 0
    boats_r = len(people) - 1
    while i <= boats_r:
        if people[boats_r] + people[i] > limit:
            boats_r -= 1
            boats += 1
        else:
            boats_r -= 1
            i += 1
            boats += 1

    return boats
```

파이썬 sort()함수의 시간복잡도는 O(n log n)
