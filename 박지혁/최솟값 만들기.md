## 최솟값 만들기
## https://school.programmers.co.kr/learn/courses/30/lessons/12941

```python
def solution(A, B):
    A.sort()                 
    B.sort(reverse=True)     
    answer = 0
    for a, b in zip(A, B):
        answer += a * b
    return answer
```
시간복잡도 : 파이썬sort O(nlogn) + 반복문O(n) = O(nlogn)
