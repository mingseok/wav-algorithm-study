## 문자열 다루기 기본

- https://school.programmers.co.kr/learn/courses/30/lessons/12918

```java
class Solution {
    public boolean solution(String s) {
        for (int i = 0; i < s.length(); i++) {
            if (s.length() != 4 && s.length() != 6) {
                return false;
            }
            
            if (!Character.isDigit(s.charAt(i))) {
                return false;
            }
        }
        return true;
    }
}
```
