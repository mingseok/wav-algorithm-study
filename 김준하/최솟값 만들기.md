## 최솟값 만들기

```java
import java.util.Arrays;

/**
 * Array는 기본적으로 sort(오름차순)만 제공하므로
 * 내림차순 정렬 시 오름차순 정렬 후 역순 배열을 작성하면 된다. (integer일 경우엔 Comparator.reverseOrder(); 로 정렬 가능)
 * 하지만 본 문제에서는 둘 다 내림차순 정렬 후 계산하면 된다.
 */

class Solution {
    public int solution(int []A, int []B) {
        Arrays.sort(A);
        Arrays.sort(B);
        int result = 0;
        for (int i=0; i<A.length; i++) result += A[i] * B[A.length-i-1];
        return result;
    }
}
```
