## 콜라스 추측

- https://school.programmers.co.kr/learn/courses/30/lessons/12945

```java
class Solution {
    public int solution(int n) {
        int firstNum = 0, secondNum = 1;
        int x = 0;
        for (int i=2; i<=n; i++) {
            x = (firstNum + secondNum) % 1234567;
            firstNum = secondNum % 1234567;
            secondNum = x % 1234567;
        }

        return x;
    }
}
```
