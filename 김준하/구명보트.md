 * Greedy 알고리즘이란 각 순간마다, 당장 눈앞에 보이는 최선의 선택을 하는 것이다.
 * 해당 문제에서 최소 구명보트를 구하기 위한 최선의 선택은 한 구명보트에 두 명을 태우는 것으로
 * 최대(최소) 몸무게를 가진 인원을 태울 때 최소(최대) 몸무게를 가진 인원을 태울 수 있는지 체크하면 된다.
 * 
 * 이런 greedy 알고리즘 문제는 개별적으로 봤을 때는 최선의 선택이지만, 전체적으로 봤을 때 적합하지 않을 수 있어 항상 체크가 필요하다

class Solution {
    public int solution(int[] people, int limit) {
        Arrays.sort(people);
        int count = 0;
        int start = 0, end = people.length-1;
        while (start <= end) {
            if (people[start] + people[end] <= limit) {
                count++;
                start++;
                end--;
            } else {
                count++;
                end--;
            }
        }
