```
class Solution {
    public String solution(String myString, String pat) {
        int lastIdx = -1;

        for (int i = myString.length() - pat.length(); i >= 0; i--) {
            if (myString.substring(i, i + pat.length()).equals(pat)) {
                lastIdx = i;
                break;
            }
        }
        
        return myString.substring(0, lastIdx + pat.length());
    }
}
```
