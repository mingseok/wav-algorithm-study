```
import java.util.*;

public class Solution {
    public int[] solution(int []arr) {
        int[] answer = {};

        Stack<Integer> stack = new Stack<>();
        
        for(Integer i : arr){
            if(stack.isEmpty()){
                stack.push(i);
            }else{
                
                if(stack.peek() != i){
                    stack.push(i);
                }else{
                    continue;
                }
                
            }
        }
            
        return stack.stream().mapToInt(Integer::intValue).toArray();
    }
}

```
반환값이 정해져 있기 때문에, 형변환에 대해서 정리를 했습니다. 
Stack<Integer> → Integer[]인 경우엔, 
Integer[] arr = s.toArray(new Integer[0]);

Stack<Integer> → int[]인 경우엔, 
int[] arr = s.stream().mapToInt(Integer::intValue).toArray();
