import java.util.*;
class Solution {
    public String solution(String[] participant, String[] completion) {
        String answer = "";

        HashMap<String, Integer> map = new HashMap<>();
        
        for(String s : participant){
            map.put(s, map.getOrDefault(s, 0) + 1);
        }
        
        for(String s : completion){
            map.put(s, map.get(s) - 1);
        }
        
        for(String s : map.keySet()){
            if(map.get(s) > 0){
                answer = s;
                break;
            }
        }
        
        return answer;
    }
}

동명이인이 있다는 조건하에 완주하지 못한 단 1명을 구하는 문제입니다.
동명이인이 있기 때문에, containsKey()가 아닌, 직접 순회하여 value 값을 비교해야 합니다.
먼저, participant 배열을 순회하여 participant라는 key 값을 기준으로 인원을 증가시켜 map에 담는 다면 key-value로 참가자를 구조화 할 수 있습니다.
이후, completion 에 포함된 이름들의 value들을 -1 을 반복하면,
최종적으로 value가 0이 아닌 단 한명을 구할 수 있습니다.

시간복잡도는 순회 3개를 포함하고 있기 때문에, O(3n)으로 계산될 수 있습니다.
