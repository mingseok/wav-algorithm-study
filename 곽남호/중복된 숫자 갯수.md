### 중복된 숫자 갯수


```java

class Solution {
    public int solution(int[] array, int n) {
        int answer = 0;
          for (int i : array) {
              if (i == n) {
                  answer++;
              }
          }
        return answer;
    }
}

//숫자 갯수을 파악하기 위해서 for문을 사용했지만 기존 for문 자체가 너무 복잡하여 오류가 계속 발생하였습니다
//검색을 통해 단순 갯수 세기에서는 for-each문을 사용하는것이 좋다고 하여 for-each문을 참고하여 작성하였습니다
//만약 for문을 사용할 경우에는 for (int i : array) 이부분이  for (int i = 0; i < array.length; i++) 이런식으로 변경됩니다.
```
