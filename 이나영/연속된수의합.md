### 연속된 수의 합(Java)
```JAVA
/**
* 연속된 수의 합(Java)
* 연속된 세 개의 정수를 더해 12가 되는 경우는 3, 4, 5입니다. 두 정수 num과 total이 주어집니다
* 연속된 수 num개를 더한 값이 total이 될 때, 정수 배열을 오름차순으로 담아 return
*/

 class Solution {
    public int[] solution(int num, int total) {
        
        int[] answer = new int[num];
        
        int mid = total / num;
        int start = mid - (num - 1) / 2;

        for (int i = 0; i < num; i++) {
            answer[i] = start + i;
        }

        return answer;
    }
}
```
