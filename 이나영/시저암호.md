## 시저암호(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/12926

```JAVA
class Solution {
    public String solution(String s, int n) {
        StringBuilder sb = new StringBuilder();

        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);

            if (ch >= 'A' && ch <= 'Z') {
                ch = (char)('A' + (ch - 'A' + n) % 26);
            } else if (ch >= 'a' && ch <= 'z') {
                ch = (char)('a' + (ch - 'a' + n) % 26);
            }

            sb.append(ch);
        }

        return sb.toString();
    }
}
```
