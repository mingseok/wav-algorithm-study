## 부족한금액계산하기(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/82612


```JAVA

class Solution {
    public long solution(int price, int money, int count) {
        long answer = 0;
        long total_fee = 0;
        
        for(long i=1; i<=count; i++){
            total_fee = total_fee + price*i;
        }
        
        answer = (total_fee > money) ? total_fee - money : 0;

        return answer;
    }
}

```

시간복잡도 : O(n)

1. for문 말고 (long)price * count * (count + 1）이 공식으로 풀면 시간복잡도 O(1) 로 연산횟수 줄일 수 있음
2. Math.max 사용으로 양수/음수 구분 가능
