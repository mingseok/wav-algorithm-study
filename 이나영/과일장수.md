## 과일장수(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/135808

```JAVA
import java.util.Arrays;

class Solution {
    public int solution(int k, int m, int[] score) {
        int answer = 0;
        
        Arrays.sort(score);

        for (int i = score.length - m; i >= 0; i -= m) {
            answer += score[i] * m;
        }
     
        return answer;
    }
}
```

시간복잡도 : O(n log n)
