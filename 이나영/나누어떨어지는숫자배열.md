## 나누어떨어지는숫자배열(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/12910

```JAVA
import java.util.*;

class Solution {
    public List<Integer> solution(int[] arr, int divisor) {
        List<Integer> list = new ArrayList<>();
        
        for(int num : arr){
            if(num % divisor == 0){
                list.add(num);
            }
        }
        
        if(list.isEmpty()){
            list.add(-1);
        }
        
        Collections.sort(list);
        
        return list;
    }
}
```
