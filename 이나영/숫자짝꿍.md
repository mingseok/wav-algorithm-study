## 숫자짝꿍(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/131128


```JAVA

class Solution {
    public String solution(String X, String Y) {
        String answer = "";
        
        int[] xArray = new int[10];
        int[] yArray = new int[10];
        
        for (int i = 0; i < X.length(); i++) {
            int num = Integer.parseInt(String.valueOf(X.charAt(i)));
            xArray[num]++;
        }
        
        for (int i = 0; i < Y.length(); i++) {
            int num = Integer.parseInt(String.valueOf(Y.charAt(i)));
            yArray[num]++;
        }
        
        StringBuilder sb = new StringBuilder();
        
        for (int i = 9; i >= 0; i--) {
            int min = Math.min(xArray[i], yArray[i]);
            
            for (int j = 0; j < min; j++) {
                sb.append(i);
            }
        }
        
        if (sb.length() == 0) {
            answer = "-1";
        } else if (sb.charAt(0) == '0') {
            answer = "0";
        } else {
            answer = sb.toString();
        }
        
        return answer;
    }
}

```

시간복잡도 : O(n + m)

1. 속도개선
Integer.parseInt(String.valueOf(X.charAt(i))); => X.charAt(i) - '0';

2. answer 변수 불필요
sb.toString();으로 return

3. 독립적 조건이면 if문 여러개로 사용 권장
