## 푸드파이트대회(Java)
## https://school.programmers.co.kr/learn/courses/30/lessons/134240


```JAVA

/**
* 푸드파이트대회(Java)
* 푸드파이트대회에서 대결은 준비된 음식들을 일렬로 배치한 뒤, 한 선수는 제일 왼쪽에 있는 음식부터 오른쪽으로, 
* 다른 선수는 제일 오른쪽에 있는 음식부터 왼쪽으로 순서대로 먹는 방식으로 진행됩니다. (중앙에는 물 배치)
* 수웅이가 준비한 음식의 양을 칼로리가 적은 순서대로 나타내는 정수 배열 food가 주어졌을 때, 대회를 위한 음식의 배치를 나타내는 문자열 return
* 
* 입력 : [1, 3, 4, 6]
* 출력 : "1223330333221"
*/

class Solution {
    public String solution(int[] food) {
        String answer = "";
        
        for(int i = 1; i < food.length; i++) {
            int cnt = food[i] / 2;
            
            for(int j = 0; j < cnt; j++) {
                answer += i;
            }
        }
        
        String reversed = new StringBuilder(answer).reverse().toString();
        
        return answer + "0" + reversed;
    }
}

```
String은 불변 객체이므로 += 연산 시 매번 새로운 문자열이 생성됨.
이 과정에서 기존 문자열을 복사하므로 누적 시간복잡도는 O(K²) (K = 최종 문자열 길이)
StringBuilder의 append()를 사용하면 내부 버퍼에 추가되므로 전체 시간복잡도는 O(K)
